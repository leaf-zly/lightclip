# Support

LightClip is maintained as a small open-source project. The fastest way to get help is to provide a clear, reproducible report.

## Where To Ask

- Bugs: open a GitHub issue using the bug report template.
- Feature requests: open a GitHub issue using the feature request template.
- Security issues: follow [SECURITY.md](SECURITY.md) and do not post sensitive details publicly.

## Before Opening An Issue

- Install the latest release from [GitHub Releases](https://github.com/leaf-zly/lightclip/releases).
- Check whether the issue already exists.
- Try restarting LightClip from the tray.
- For packaged app issues, mention whether you used the installer or portable build.

## Useful Diagnostics

Include:

- LightClip version.
- Windows version.
- Installer or portable build.
- Whether startup, image history, file history, or a custom shortcut is enabled.
- Reproduction steps.
- Screenshots or short recordings for UI issues.

Do not attach real clipboard history, secrets, tokens, passwords, private screenshots, or personal file paths unless you have redacted them.
