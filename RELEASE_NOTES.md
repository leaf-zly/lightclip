# LightClip v0.1.0

LightClip 首个可用版本，面向 Windows 10/11 的轻量剪贴板历史工具。

## 新功能

- 文本剪贴板历史：自动记录、去重、搜索、固定、删除和清空未固定记录。
- 图片历史：默认关闭，可在设置中开启，用于记录截图或图片剪贴板。
- 文件历史：默认关闭，可记录资源管理器复制的文件路径，并尽量恢复为 Windows 文件剪贴板。
- 自定义标题栏：包含应用图标、标题、「文件 / 编辑 / 视图 / 窗口」菜单和窗口控制按钮。
- 托盘常驻：关闭窗口默认隐藏到托盘，支持从托盘菜单退出或打开数据目录。
- 开机自启：可在设置中打开当前用户登录后自动启动。
- 快捷键唤起：默认 `Alt + V` 显示或隐藏 LightClip。
- 本地隐私：历史数据仅写入本机 Electron `userData` 目录。

## 下载

- `LightClip Setup 0.1.0.exe`：安装版，适合长期使用。
- `LightClip 0.1.0.exe`：便携版，下载后直接运行。

## 注意事项

- 当前版本未做代码签名，Windows 首次运行可能提示未知发布者。
- 图片历史和文件历史涉及敏感内容，默认关闭，建议按需开启。
- 文件历史恢复为文件剪贴板依赖 Windows PowerShell STA Clipboard API；受系统策略限制时会回退为路径文本。
